# lyric-lab — 产品需求

## 产品一句话
日语歌词学习助手：输入歌名（支持模糊搜索）→ API 自动获取歌词 + 罗马音 + 中文翻译 + 单词拆解 → 逐句/逐词听发音、学理解，顺便积累日语词汇。

注：本工具不处理歌曲音频。用户在 QQ 音乐等平台听歌，用本工具看歌词 + 听发音学语言。发音朗读使用浏览器内置日语 TTS（免费、离线可用）。

## 目标用户
- 想学唱日语歌但不会日语的人
- 希望在唱歌过程中自然积累日语词汇的学习者
- 使用手机练习（PWA），偶尔 PC

## 核心用户故事

| 故事 | 优先级 |
|------|--------|
| 输入歌名（支持模糊搜索），自动获取日文歌词 + 罗马音 + 中文翻译，三行对照展示 | P1 |
| 每句有发音按钮，点击用日语 TTS 朗读该句，能反复听 | P1 |
| 点击歌词中的单词，看到读音、释义、词类标注，也能单独听该单词发音 | P1 |
| 把歌词中的单词加入生词本，后续用卡片复习 | P2 |
| 按句分段浏览，对不熟的句子反复看、反复听 | P1 |
| 多首歌管理，已解析的歌曲离线可查看 | P1 |

## 功能清单

### P1 — MVP 必须要有
- [ ] **歌曲搜索**：输入歌名（支持模糊、不完整的关键词），调用 API 搜索匹配的日语歌曲，返回候选列表
- [ ] **歌词获取与解析**：选定歌曲后，API 一次性返回：日文歌词原文 + 罗马音 + 中文翻译 + 逐词拆解（含词类标注）
- [ ] **歌词三行对照**：每句展示日文 / 罗马音 / 中文翻译，按句分段，滚动浏览。每句有 ▶ 发音按钮
- [ ] **TTS 发音**：使用浏览器 SpeechSynthesis API（日语语音），每句和每个单词都能点读发音
- [ ] **单词拆解标注**：每句中的单词可点击，弹出释义气泡（读音、释义、日常高频词 or 歌词专用表达），气泡内也有发音按钮
- [ ] **数据持久化**：歌曲、歌词、单词数据存 IndexedDB，离线可查看已缓存的歌曲（TTS 依赖浏览器，离线也可用）

### P2 — 下一迭代
- [ ] **生词本**：从歌词中收藏单词，按歌曲分组，查看所有收藏
- [ ] **单词卡片复习**：正面日文 + 罗马音，反面中文释义，简单翻卡模式
- [ ] **手动补充歌词**：如果 API 搜不到歌词，允许用户手动粘贴日文歌词文本，然后 API 解析

### P3 — 未来
- [ ] **音频支持**：导入本地 MP3，歌词 + 音频联动（跳播、循环）
- [ ] **录音跟唱**：录音 + 与原唱对比
- [ ] **时间轴同步**：LRC 支持，歌词高亮跟随音频进度
- [ ] **音高反馈**：复用 harmonica-tone 的音高检测
- [ ] **学习统计**：练习时长、学会的单词数、连续打卡

## 交互行为

### 歌曲搜索与导入
1. 用户打开 app，看到搜索框（placeholder：「输入歌名，支持模糊搜索」）
2. 输入关键词（如「残酷」「eva」「unravel」），实时或点击搜索
3. 显示匹配的候选歌曲列表（歌名 + 艺人）
4. 点击某首歌 → 调用 API 解析歌词 → 显示进度 → 完成后进入歌词练习界面

### 练习界面
- 顶部：歌曲标题 + 艺人 + 返回按钮
- 主体：歌词列表，每句三行——
  - 第一行：日文原文（较大字号）+ 句末 ▶ 发音按钮
  - 第二行：罗马音（灰色，可隐藏/显示切换）
  - 第三行：中文翻译（小字，可隐藏/显示切换）
- 单词交互：点击日文中的单词 → 底部弹出 sheet（读音 + 释义 + 词类标签 + 🔊听发音 + 「加入生词本」按钮）
- 底部工具栏：显示/隐藏罗马音 · 显示/隐藏翻译 · 字号调节
- TTS 说明：使用浏览器内置日语语音合成，免费，支持离线

### 生词本（P2）
- 按歌曲分组展示收藏的单词
- 每个单词显示：日文 + 读音 + 释义 + 来源歌曲
- 复习模式：翻卡式，点一下翻转显示释义

## 非功能需求

- **PWA**：可安装到手机主屏幕，manifest.json + Service Worker
- **离线能力**：已解析的歌曲数据完全离线可用（IndexedDB）；搜索和新歌解析需联网
- **API 成本**：搜索 + 解析一首歌预计消耗 2000-5000 token，费用极低；仅在导入新歌时调用
- **API 复用**：复用 text-reader 的 DeepSeek API key（localStorage 共享或独立配置）
- **响应式**：手机优先设计，最大宽度 480px 居中，PC 也能用
- **单文件**：遵循仓库惯例，index.html 包含所有 HTML + CSS + JS
- **暗色主题**：与 metronome、harmonica-tone、text-reader 风格一致
- **存储**：IndexedDB 存歌曲/歌词/单词数据；localStorage 存设置 + API key

## 竞品参考
- LyricsTraining（填空式练歌词，侧重听力，不侧重发音和单词学习）
- 各种卡拉 OK app（侧重娱乐，不侧重语言学习）
- Musixmatch（歌词同步，无学习功能）
- MaruMaru / 歌詞GET（日语歌词搜索，无逐词拆解和学习功能）
- 本工具的差异化：**歌词获取 + 三行对照 + 逐词拆解 + 生词收藏复习** 一体化

## 不做的事（V1）
- 不处理音频（不导入 MP3、不播放、不录音）
- 不支持多语言（仅日语）
- 不支持时间轴同步
- 不支持视频
- 不支持社交分享
- LRC 文件导入（留到 P3）
