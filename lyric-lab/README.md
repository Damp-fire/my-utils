# Lyric Lab — 日语歌词学习助手

## 是什么
输入日语歌名 → 自动获取歌词（罗马音 + 中文翻译 + 逐词拆解）→ 听发音 → 收藏单词，边唱歌边学日语。

## 怎么用

1. 在手机浏览器打开 `index.html`
2. 设置 → 填入 DeepSeek API Key
3. 搜索歌名（支持模糊搜索，如「残酷」「eva」）
4. 点击候选歌曲 → 自动生成三行对照歌词
5. 每句末尾 ▶ 听发音，点击单词查看释义
6. 收藏的单词在「生词本」查看

## 运行方式
- 无需安装，直接用浏览器打开 `index.html`
- 支持 PWA 安装到手机主屏幕
- 已解析的歌曲离线可用

- [ ] 搜索「残酷」→ 看到候选列表

- [ ] 点击单词 → 底部弹出释义（读音、释义、词类标签）

- [ ] 生词本按歌曲分组展示
- [ ] 罗马音 / 翻译可切换显隐
- [ ] 设置页可配置 API 端点、Key、模型

- [ ] PWA 可安装到主屏幕
- [ ] 离线打开已缓存的歌曲

## 技术栈
- 单文件 HTML + CSS + JS（零依赖）
- IndexedDB 本地存储
- DeepSeek API（歌词搜索 + 解析）
- 浏览器 SpeechSynthesis（日语 TTS）
- PWA（Service Worker + manifest.json）
